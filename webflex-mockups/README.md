# Webflex Mockups

Static host for per-prospect cold-outreach mockups generated by the
`map-scraper` pipeline.

Tokens land at `https://mockups.webflex.cz/<token>/` (custom domain)
or `https://<github-user>.github.io/webflex-mockups/<token>/` (default).

Each token directory holds:
- `index.html` — the mockup page
- `preview.png` — thumbnail used as inline hero in the cold email

Mockups carry `<meta name="robots" content="noindex">` and a footer
disclaimer; they auto-rotate as new prospects join. Delete a token
directory + push to retire it.
